## SIMCONNECT\_DATA\_PBH

The **SIMCONNECT\_DATA\_PBH** structure is used to hold a world orientation.

##### Syntax

```cpp
struct SIMCONNECT_DATA_PBH{
    float Pitch;
    float  Bank;
    float  Heading;
    };
```

##### Members

| Member | Description |
| `Pitch` | The pitch in degrees. |
| `Bank` | The bank in degrees. |
| `Heading` | The heading in degrees. |
