SIMCONNECT\_STATE

## SIMCONNECT\_SIMOBJECT\_TYPE

The **SIMCONNECT\_STATE** enumeration type is used with the [SimConnect\_SetSystemEventState](../General/SimConnect_SetSystemEventState.htm) call to turn the reporting of events on and off.

##### Syntax

```cpp
enum SIMCONNECT_STATE{
    SIMCONNECT_STATE_OFF,
    SIMCONNECT_STATE_ON
    };
```

##### Members

| Member | Description |
| `SIMCONNECT_STATE_OFF` | Specifies off. |
| `SIMCONNECT_STATE_ON` | Specifies on. |

##### Remarks

See the remarks for [SimConnect\_MapInputEventToClientEvent](../InputEvents/SimConnect_MapInputEventToClientEvent.htm).
