## SimConnect\_UnsubscribeToFacilities

The **SimConnect\_UnsubscribeToFacilities** function is used to request that notifications of additions to the facilities cache are not longer sent.

##### Syntax

```cpp
HRESULT SimConnect_UnsubscribeToFacilities(
    HANDLE  hSimConnect,
    SIMCONNECT_FACILITY_LIST_TYPE  type
    );
```

##### Parameters

| Parameter | Description | Type |
| --- | --- | --- |
| hSimConnect | Handle to a SimConnect object. | Integer |
| type | Specifies one member of the [`SIMCONNECT_FACILITY_LIST_TYPE`](../Structures_And_Enumerations/SIMCONNECT_FACILITY_LIST_TYPE.htm) enumeration type. | Enum |

##### Return Values

The function returns an **HRESULT**. Possible values include, but are not limited to, those in the following table.

| Return value | Description |
| --- | --- |
| S\_OK | The function succeeded. |
| E\_FAIL | The function failed. |

##### Remarks

This is used to terminate notifications generated by the [`SimConnect_SubscribeToFacilities`](SimConnect_SubscribeToFacilities.htm) function.
